## Data by XFOIL Polar

| Column  | Description                              |
|---------|------------------------------------------|
| Alpha   | angle of attack (°)                      |
| Cl      | lift coefficient                         |
| Cd      | drag coefficient                         |
| Cdp     | pressure (viscous) drag                  |
| Cm      | moment coefficient (about ¼ chord)       |
| Top_Xtr | transition location (x/c) for upper side |
| Bot_Xtr | transition location (x/c) for lower side |
